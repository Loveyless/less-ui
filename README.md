<h1 align="center">🎉Welcome to Less UI🎉</h1>


## 介绍 📖

Vue3 UI 组件库(demo)
使用 Less UI 快速搭建你的 Vue3 项目

## 使用 🔨

建设中 见指令文档

## 指令 📚

执行```pnpm dev```即可启动开发测试环境(root/src/main.ts中测试)
执行```pnpm build```即可打包代码到root/dist中(root/demo中测试)
执行```pnpm docs:dev```指令 即可在本地启动组件库文档
